# arc-core (DEPRECATED)

**Status:** Secondary / duplicate — do not land new product work here.

**Canonical extension:** `packages/arc-extension`

This package was extracted from the monolithic extension during modularization.
`packages/arc-extension` is now the canonical, tested, and wired Arc Studio Theia
extension. This package is retained during the transition period so existing UI
widgets continue to function, but no new features or bug fixes should target
this package.
