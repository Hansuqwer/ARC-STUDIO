# TUI Next Slice: Streaming Fix + CommandPalette + Settings + HelpScreen + Secret Scan

## Context

ARC Studio v0.8-r-ux2. Five TUI gaps discovered by comprehensive audit:

1. Streaming tokens are invisible at the widget layer.
   append_to_last() mutates DataStore.entries[-1].content but the
   already-mounted MarkdownBlock does not re-render. Streamed messages
   appear only when the NEXT entry is added, not in real time.

2. CommandPalette is empty when opened before SlashCommandHandler is
   instantiated. CommandPalette.on_mount() calls get_registry() which
   returns the bare (empty) singleton. SlashMenu calls _build_registry()
   which populates it. Different call paths → inconsistent state.

3. SettingsView only shows dark/light in the theme RadioSet (missing
   mocha/latte/high-contrast/mono). Also, theme and mode changes are NOT
   persisted on Apply — only isolation backend is saved.

4. HelpScreen is static hardcoded text missing 12+ implemented features:
   Ctrl+X, Shift+Tab, /plan, /build, /auto, /review, /budget, /models,
   /wallet, /providers, /title, /statusline.

5. Shell escape stdout is shown unredacted. If a user runs `cat ~/.aws/credentials`
   the credentials appear in the transcript. redaction_applied=False in all
   audit events confirms this is a known gap.

## Scope

### 1. Fix streaming: subscribe MarkdownBlock to DataStore changes

File: python/src/agent_runtime_cockpit/tui/widgets/transcript.py

In the Transcript polling loop (100ms interval), after mounting a new
entry widget, also check if the LAST entry is being streamed:

```python
def _poll_entries(self):
    # ... existing new-entry mounting logic ...

    # If streaming, refresh the last assistant MarkdownBlock
    if self.data.is_streaming and self.data.entries:
        last = self.data.entries[-1]
        if last.role == "assistant":
            # Find the last mounted MarkdownBlock and update its content
            blocks = self.query(MarkdownBlock)
            if blocks:
                last_block = blocks.last()
                last_block.update(last.content)
```

Where `MarkdownBlock.update(text)` calls `self.update(Markdown(text))` or
`self.renderable = Markdown(text)` + `self.refresh()`.
