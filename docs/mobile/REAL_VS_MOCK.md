# ARC Mobile Runtime — Real vs Mock Matrix

ARC Mobile Runtime is currently a **simulator preview**. The implemented runtime is safe, local, static/mock-oriented infrastructure for planning, policy, simulation, and trace inspection. It is **not** a production native mobile SDK yet.

## Current truth

- No production native iOS or Android bridge exists.
- No real camera, microphone, contacts, calendar, photos, files, health, or location access exists.
- No autonomous background execution exists.
- No production MCP gateway exists.
- Built-in capabilities are mock/simulator capabilities.
- CLI simulation predicts and records what a plan would do; it does not execute native device APIs.

## Hardening modules (Batch 6)

These are **real, local, deterministic** infrastructure modules. They harden the simulator preview's data-handling and policy surfaces. None of them grant real device access — the capability entry-gate is default-denied and always routes native capabilities to fixtures.

| Module / CLI | What it does | Classification | Not |
|---|---|---|---|
| `secure_store` · `arc mobile secure-store` | Fernet encryption-at-rest with sensitivity classification; CLI output is redacted (value never printed) | Real local crypto-at-rest | Not OS Keychain/Keystore-backed |
| `privacy_budget` (EgressGuard) · `arc mobile egress check` | Deterministic byte-budget + per-class egress decisions; critical class blocked | Real deterministic guard | Not a network interceptor |
| `offline_queue` · `arc mobile queue` | Durable hash-only queue (SHA-256 + metadata, no raw payload), TTL, FIFO | Real local queue | Not a sync/transport layer |
| `feature_flags` · `arc mobile flags` | Default-off flags + kill switch with path persistence | Real local flags | Not a remote-config service |
| `capability_gate` · `arc mobile gate evaluate` | Default-denied capability entry-gate; routes to fixtures | Real deterministic gate | Does not unlock real device access |
| `policy_context` (TenantPolicyHook) · `arc mobile policy explain --org-bundle` | Signed org bundle RBAC/ABAC overlay; fails closed on bad signature/role/tenant | Real deterministic overlay | Advisory only; not tenant isolation |
| `siem_export` · `arc mobile siem-export` | Export simulated trace to CEF/JSON for SIEM | Real export adapter | Not a live SIEM connector |
| `sbom` · `arc mobile sbom` | CycloneDX SBOM of the mobile runtime surface | Real SBOM generator | Static inventory only |
| `audit_retention` · `arc mobile audit-retention` | TTL/count retention + size rotation on the decisions log | Real local retention | Not a signed production audit log |
| `mcp_bridge` | Default-off loopback dev bridge guard (token + TTL, fail-closed) | Real default-off guard | Not a production MCP gateway |
| `compliance/report` · `arc mobile generate compliance-report` | Aggregated compliance posture report | Real local report | Not a certification |

## Naming rule

Until real mobile framework bridges, signed plans, scoped approvals, compliance artifacts, and production traces exist, refer to this as:

**ARC Mobile Runtime Simulator Preview**

Avoid calling it production-ready, enterprise-ready, or a real native mobile SDK.
