# ADR-012: Provenance Wedge Framing for ARC Studio Scope

Status: Rejected — superseded by ADR-011
Date: 2026-05-19
Rejected: 2026-05-19
Deciders: ARC Studio core team
Related: ADR-011 (Accepted)

## Context

The provenance wedge framing considered narrowing ARC Studio scope around audit/provenance first, deferring full parity with general-purpose agent CLIs and IDEs.

## Decision

Rejected. ARC Studio adopts ADR-011 full parity framing instead.

## Rejection Note

This ADR was rejected on 2026-05-19 in favor of ADR-011 (Full Parity
Framing). The team committed to full competitive parity with Claude Code,
OpenCode, and Codex CLI while retaining ARC's audit/provenance/gating
differentiators as additive strengths.

This document is preserved in the repository as the historical record of
the alternative that was considered. Future re-evaluation of scope should
read both ADRs together with an honest assessment of execution capacity at
that time. A return to the wedge framing would require a new ADR
explicitly superseding ADR-011.
