## 4. Task-by-task

### Task 1 — Pick persistence backend (~15 min decision)

Per `docs/research/budget-persistence-audit.md` §4, the candidates are:

| Pattern | Pros | Cons | Recommend |
|---|---|---|---|
| **SQLite WAL** | Crash-safe (WAL survives SIGKILL), multi-process safe (WAL+advisory lock), idiomatic for local-first | One extra dep (`sqlite3` is stdlib, so actually zero deps), schema migrations needed eventually | **YES — default choice** |
| **Append-only JSONL + fsync + flock** | Human-readable, crash-safe (append+fsync), auditable | Slower reads as ledger grows; needs periodic compaction; flock has known cleanup edge cases | Fallback if SQLite ruled out |
| Single JSON + `os.replace` | Simple | Bad concurrency; only OK if single-process by design | **REJECTED** — concurrency unsafe |

**Decision:** Use **SQLite WAL**. Rationale:
- Single-user local-first matches SQLite's sweet spot
- WAL mode survives SIGKILL with no data loss
- Stdlib (no new dependency)
- Schema migrations stay simple — we have one table for foreseeable future
- ~50 LOC less than JSONL+flock cleanup logic

Storage path:
- macOS: `~/Library/Application Support/arc-theia-studio/budget.db`
- Linux: `~/.local/share/arc-theia-studio/budget.db`
- Windows: `%LOCALAPPDATA%/arc-theia-studio/budget.db`

Use `platformdirs` if already in deps; otherwise use a small `_data_dir()` helper inline. Verify:

### Task 2 — Implement persistence (~4-6h)

#### §A. Storage interface

```python
# python/src/agent_runtime_cockpit/budget/storage.py
from __future__ import annotations

import abc
import sqlite3
from decimal import Decimal
from pathlib import Path
from typing import Mapping

from .schema import BudgetScope


class BudgetStorage(abc.ABC):
    """Abstract budget-state persistence layer.

    Implementations MUST be:
      - crash-safe (mid-write SIGKILL preserves last consistent state)
      - concurrent-safe (multiple processes correctly accumulate)
      - fail-closed (unreadable state → raise, don't return zero silently)
    """

    @abc.abstractmethod
    def load_spent(self) -> Mapping[tuple[BudgetScope, str | None], Decimal]:
        """Returns map of (scope, scope_key) → accumulated USD spent.
        scope_key: provider_id for PROVIDER_DAY (with date window),
                   session_id for SESSION, None otherwise."""

    @abc.abstractmethod
    def add_spend(
        self,
        scope: BudgetScope,
        scope_key: str | None,
        amount_usd: Decimal,
    ) -> None:
        """Append a spend record atomically."""

    @abc.abstractmethod
    def reset_scope(self, scope: BudgetScope, scope_key: str | None) -> None:
        """Used by RUN scope on run completion (in-memory case ignores)."""


class InMemoryStorage(BudgetStorage):
    """Default. Preserves v0.4.0-alpha behavior — no persistence."""
    def __init__(self) -> None:
        self._spent: dict[tuple[BudgetScope, str | None], Decimal] = {}

    def load_spent(self): return dict(self._spent)
    def add_spend(self, scope, scope_key, amount):
        key = (scope, scope_key)
        self._spent[key] = self._spent.get(key, Decimal(0)) + amount
    def reset_scope(self, scope, scope_key):
        self._spent.pop((scope, scope_key), None)


class SQLiteWALStorage(BudgetStorage):
    """Crash-safe + multi-process safe persistence via SQLite WAL mode."""

    SCHEMA = """
    CREATE TABLE IF NOT EXISTS spend_log (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        ts TEXT NOT NULL,                    -- ISO8601 UTC
        scope TEXT NOT NULL,                 -- RUN | WORKFLOW | SESSION | PROVIDER_DAY
        scope_key TEXT,                      -- provider_id|session_id|NULL
        amount_usd TEXT NOT NULL             -- Decimal as string
    );
    CREATE INDEX IF NOT EXISTS ix_spend_scope_key
      ON spend_log (scope, scope_key);
    CREATE TABLE IF NOT EXISTS schema_meta (
        version INTEGER PRIMARY KEY
    );
    INSERT OR IGNORE INTO schema_meta (version) VALUES (1);
    """

    def __init__(self, db_path: Path) -> None:
        db_path.parent.mkdir(parents=True, exist_ok=True)
        self._conn = sqlite3.connect(db_path, isolation_level=None)
        self._conn.execute("PRAGMA journal_mode=WAL;")
        self._conn.execute("PRAGMA synchronous=NORMAL;")  # WAL-safe
        self._conn.executescript(self.SCHEMA)

    def load_spent(self):
        cur = self._conn.execute(
            "SELECT scope, scope_key, SUM(amount_usd) FROM spend_log "
            "GROUP BY scope, scope_key"
        )
        out = {}
        for scope_name, scope_key, total in cur:
            scope = BudgetScope[scope_name]
            # PROVIDER_DAY scope_key encoded as 'provider_id|YYYY-MM-DD'; filter
            # to today only
            if scope == BudgetScope.PROVIDER_DAY and not _is_today(scope_key):
                continue
            out[(scope, scope_key)] = Decimal(total)
        return out

    def add_spend(self, scope, scope_key, amount):
        from datetime import datetime, timezone
        self._conn.execute(
            "INSERT INTO spend_log (ts, scope, scope_key, amount_usd) "
            "VALUES (?, ?, ?, ?)",
            (datetime.now(timezone.utc).isoformat(), scope.name, scope_key, str(amount)),
        )

    def reset_scope(self, scope, scope_key):
        self._conn.execute(
            "DELETE FROM spend_log WHERE scope=? AND (scope_key=? OR (? IS NULL AND scope_key IS NULL))",
            (scope.name, scope_key, scope_key),
        )


def default_storage(persistent: bool = True) -> BudgetStorage:
    """Factory used by BudgetEnforcer when no explicit storage passed."""
    if not persistent:
        return InMemoryStorage()
    # platformdirs-derived path; see §B
    return SQLiteWALStorage(_data_dir() / "budget.db")
```

def test_run_scope_resets_per_enforcer_instance(tmp_path):
    db = tmp_path / "budget.db"
    e1 = BudgetEnforcer(storage=SQLiteWALStorage(db))
    e1.record_spend(BudgetScope.RUN, Decimal("0.50"))
