# v0.5.1-alpha — Chinese-Labs / Open-Weight Vendor Adoption

> **Release type:** feature
> **Scope:** Route to DeepSeek, Qwen, Kimi, GLM, MiMo, and MiniMax via the
> existing `providers/openai_compatible.py` adapter. No new provider
> *classes* required; per-vendor *cost rows* with vendor-specific cache
> behavior, free-tier flags, and tokenizer warnings. Same persistence +
> wallet infra from v0.4.1; same compaction + handles from v0.5.0.
> **Source:** `docs/research/pricing-snapshot-2026Q2-addendum-chinese-labs.md`
> (the entire addendum is this spec's planning document).
